      Grading Computation Matrix

      1. Overall Rating Equation

      The final rating is derived using a weighted sum model scaled to a maximum of 5.0:

      Rfinal = (CSacad × 0.10) + (CSlead × 0.25) + (CSpat × 0.25) + (CSsoft × 0.25) + (CStech × 0.15)

      2. Component Score (CS) Matrices

      Academic (CSacad)

      Cadets must provide their GWA by summing up the other major and minor courses, but they must NOT include the competency appraisal yet. Note: 1st semester is different from 2nd semester, then 3rd semester, etc. Evaluated independently per semester.

      • GWA ≥ 90.0 → 5.0
      • 85.0 ≤ GWA < 90.0 → 4.0
      • GWA < 85.0 → 3.0
      Leadership (CSlead)
      • Corps of Midshipmen Officer → 5.0
      • Standard Cadet → 3.0
      Patriotism (CSpat)

      Ratio (P) = Cum. Merits ÷ Benchmark Merits

      • P ≥ 0.75 → 5.0
      • 0.50 ≤ P < 0.75 → 4.0
      • 0.25 ≤ P < 0.50 → 3.0
      • 0.01 ≤ P < 0.25 → 2.0
      • P < 0.01 → 1.0
      Technical Core (CStech)

      Ratio (T) = Passed Tech Tasks ÷ Total Tech Tasks

      • T ≥ 0.90 → 5.0
      • 0.80 ≤ T < 0.90 → 4.0
      • 0.70 ≤ T < 0.80 → 3.0
      • 0.60 ≤ T < 0.70 → 2.0
      • T < 0.60 → 1.0

      3. Soft Skills & Watchkeeping Integration

      The Soft Skills score (CSsoft) is a hybrid metric averaging the completion rate of the Soft Tasks Checklist and the Daily Logbook Submission Rate for active graded months.

      CSsoft = ⌈ (Soft Task Rating + Logbook Rating) ÷ 2 ⌉

      4. Bonus Merits & Star Cadets

      Star Cadet System: The qualification for a Star Cadet is they must gain a 90+ GWA on their current semester. Earning this status grants an automatic perfect 5.0 Academic CS and a +2% bonus to their final specific grade.

      Individual Extra Merits: Instructors can manually assign an Individual Extra Bonus Percentage for extraordinary performance. This percentage is directly added to the final specific grade.

      Final Specific Grade = Base % Grade + Star Bonus (if applicable) + Individual Bonus %
